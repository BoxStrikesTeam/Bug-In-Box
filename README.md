# Box Strike Vulnerability Database

> A public knowledge base of responsibly disclosed security vulnerabilities researched by the Box Strike Team.

## About

This repository contains security vulnerabilities discovered by the **Box Strike Team**.

Our goal is to document real-world vulnerabilities, share technical knowledge with the security community, and maintain a transparent archive of responsibly disclosed findings.

Every published report may include technical details, impact analysis, proof-of-concept (when permitted), remediation information, and disclosure timeline.

---

## Responsible Disclosure Policy

We strictly follow responsible disclosure practices.

* Newly discovered vulnerabilities are **never published immediately**.
* Every finding is first reported to the affected vendor or organization.
* Reports remain **private** throughout the disclosure process.
* A vulnerability is added to this repository **only after it has been resolved, patched, or officially disclosed** in accordance with the vendor's disclosure policy.

This approach helps protect users while promoting responsible security research.

---

## Publication Criteria

A report is published only when one or more of the following conditions have been met:

* The vulnerability has been fixed.
* The vendor has completed the disclosure process.
* Public disclosure has been authorized.
* Publication complies with the applicable bug bounty or vulnerability disclosure policy.

Any report that does not satisfy these conditions will remain private.

This repository exists to:
